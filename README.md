# my-portfolio
# 🚀 Shristy | Beginner Frontend Portfolio

> My very first web development project built during my early learning phase in college.

---

## 📖 Project Story

This portfolio website represents the foundation of my journey into web development.

When I started building this project, I had very little knowledge about how websites actually worked. I was just beginning to learn HTML, CSS, and JavaScript. Through experimentation, debugging, and consistent practice, I gradually understood how structure, styling, and interactivity come together to create a complete web experience.

This project marks the starting point of my frontend development journey and reflects my growth mindset as a learner.

---

## 🎯 Purpose of the Project

- To apply the fundamentals of web development
- To understand responsive layout structure
- To practice DOM manipulation using JavaScript
- To build a personal brand presence for internship opportunities
- To gain hands-on experience through real implementation

---

## 🛠 Tech Stack

- **HTML5** – Structure of the website  
- **CSS3** – Styling and responsive layout  
- **JavaScript** – Interactivity and DOM manipulation  

---

## ✨ Features Implemented

✔ Responsive layout  
✔ Smooth scrolling  
✔ Dark / Light theme toggle  
✔ Scroll-to-top button  
✔ Clean and structured UI  
✔ Organized code structure  

---

## 🧠 What I Learned

- Writing semantic HTML
- Managing layouts using CSS
- Handling user interactions with JavaScript
- Understanding DOM manipulation
- Improving debugging skills
- Building confidence through practical implementation

---

## ⚡ Challenges Faced

- Managing layout alignment issues
- Handling theme state properly
- Understanding event listeners
- Fixing small but impactful UI bugs

Each challenge helped me improve my logical thinking and problem-solving ability.
